<h2>Usage:</h2>
<h3>Single command</h3>

```bash
curl https://raw.githubusercontent.com/FCsab/Basic-Linux-Setup-Script/main/setup.sh >> setup.sh && chmod +x setup.sh && sudo bash setup.sh
```
